# SKSU Voting System

This system is developed for Sultan Kudarat State University.

## Features
- Role-based login (Admin, OSAS, Student)
- OSAS CRUD for positions and candidates
- Student voting system
- Secure authentication
- One vote per position

## Technologies Used
- PHP
- MySQL
- HTML/CSS
- XAMPP

## How to Run
1. Import database.sql into phpMyAdmin
2. Configure db.php
3. Run via localhost
